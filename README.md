# SkyCraft Launcher

SkyCraft Launcher — лаунчер для Minecraft, разработанный для серверных сборок SkyCraft.

## Возможности

* Автоматическая установка Java, Minecraft и загрузчиков (NeoForge, Forge, Fabric).
* Установка и обновление сборок.
* Проверка файлов по SHA-256.
* Поддержка нескольких сборок.
* Пользовательские моды.
* Настройка пути установки и объёма оперативной памяти.
